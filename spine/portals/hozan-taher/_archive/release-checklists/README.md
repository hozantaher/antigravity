# release-checklists

Historické release checklisty pro dokončené releases. Jsou to snapshoty toho, co se ověřovalo v době GA, ne živé procesy.

## Obsah

- `v0.1.0-GA-CHECKLIST.md` — anti-trace-relay + privacy-gateway v0.1.0 GA checklist. Released commit `c24a890` "Release v0.1.0 GA". Stále existují jako aktivní služby; checklist byl pro tamtu specifickou verzi.

## Kdy sem přidávat

Po každém GA release, pokud checklist obsahuje specifické kroky co budou jinak v další verzi (je to lessons-learned artifact). Pokud je checklist generický a dál použitelný → nechat v `docs/playbooks/`.
