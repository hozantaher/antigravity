# MCP Nástroje (Tools)

Zde je seznam nástrojů, které poskytuje náš nativní MCP server.

- `antigravity_resolve_node`: Načte kontext uzlu.
- `antigravity_audit_drift`: Spustí Cybernetic Governor.
- `antigravity_rename_node`: Provede bezpečný refaktoring.
- `antigravity_create_node`: Generátor kódu.
