# Architektura Rozhodnutí: 002 - Magic Comments místo AST

## Status
Zavedeno.

## Rozhodnutí
Pro reverzní linkování z externích repozitářů (karantény) do byznysové páteře používáme magické komentáře ve formátu `// @vektor-link: node-name`, místo toho abychom se snažili složitě analyzovat AST a importy.

*Detaily budou doplněny.*
